Для работы программы необходимо заполнить настройки в файле option.json.

Общие настройки:
1. locationWS : Адресс веб-сервиса для получения маршрутов.

Пользовательские настройки:
2. printerName : Имя принтера на котором будет производиться печать.

3. AcroRd32exe : Путь к файлу AcroRd32.exe*

* Для работы программы должен быть установлен Adobe Acrobat Reader. Скачать и установить можно
по ссылке https://get.adobe.com/ru/reader/
ВНИМАНИЕ! В пути должн быть использован прямой слэш "C:/Program Files (x86)/Adobe/Acrobat Reader DC/Reader/AcroRd32.exe"
